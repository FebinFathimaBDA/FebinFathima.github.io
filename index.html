Big Data

Definition:
Big Data is a collection of diverse, complex and reliable data, yet to grow exponentialy by time in hypervelocity which are impossible to process using traditional data management tools and methods. It performs analysis, processing and storage of reliable data including both structured and unstructured data at real-time from heterogenous sources. These processed data is visualized to learn trends and patterns and helps organizations in better decision making, improved customer satisfaction and improved efficiency in operations.

Key Properties of Big Data "4 Vs"

Variety, Velocity , Volume & Veracity are the key properties of Big Data. A massive volume of data can be collected and processed using big data processing tools compared to traditional data analysis tools. Variety means different types of data being generated including structured like texts or databases and unstructured like videos, images, xml files etc. The term velocity refers the speed at which data is continuously stored, analysed, and processed at real-time. The last V, Veracity refers to clean and reliable data which is very important for decision making.

Big Data analytics methods includes Real-Time analytics, Distibuted computing using tools like Hadoop & spark,predictive analytics which uses machine learning algorithms to predict future trends, NoSQL databases which can store unstructured datas, Machine vision and Natural Language Processing methods.
